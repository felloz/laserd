## About The Project

[![Product Name Screen Shot][product-screenshot]](https://raw.githubusercontent.com/othneildrew/Best-README-Template/master/images/screenshot.png)

I'm just a programmer with some time who decided to get into the workings of development frameworks. Based on information from other frameworks and libraries, I decided to create a "new" one with the libraries that are more to my liking. Currently I only dedicate my free time to work on it, however so far I have made progress in an MVC (Many improvements remain, but I will incorporate them later)

Additionally, I have incorporated Vue.js in its version 2.0 and I am sure that you will like it.

You are free to use it if you like and complete that part that lies ahead.
